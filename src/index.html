Field est un jeu 2D de type “plates-formes réflexion” dans lequel le joueur est confronté à un environnement qu'il devra déjouer afin d'atteindre la sortie.

Le gameplay est construit autour de la coopération de deux joueurs qui incarnent deux personnages. L'un maîtrise les champs magnétiques, l'autre peut changer la constitution de son corps métallique. Mais un mode solo est également disponible !

Parcourez la galerie ou la section "à propos" pour en découvrir plus sur le jeu, son univers et son équipe.

Télécharger

En chantier !

Vous pourrez bientôt profiter d'une galerie de screeshots !

Windows

client (.zip, 100Mo) serveur (.zip, 3Mo)

Procédure d'installation :

  • Récupérer le client et le serveur (fichiers .zip)
  • dézipper le dossier contenant le serveur
  • dézipper le dossier contenant le client
  • lancer le serveur (pas nécessaire en mode solo)
  • lancer le client

Mac OS X

client (.zip, 100Mo) serveur (.zip, 12Mo)

Procédure d'installation :

  • Récupérer le client et le serveur (fichiers .zip)
  • dézipper le dossier contenant le serveur
  • dézipper le dossier contenant le client
  • lancer le serveur (pas nécessaire en mode solo)
  • lancer le client

GNU/Linux

client (.love, 100Mo) serveur (.love, 3Mo)

Procédure d'installation sous ubuntu :

  • Récupérer le client et le serveur (fichiers .love)
$ sudo add-apt-repository ppa:bartbes/love-stable
$ sudo apt-get update
$ sudo apt-get install love
$ love field_serveur.love # (pas nécessaire en mode solo)
$ love field_client.love
